Using Knockoutjs's foreach on nested jQuery Mobile lists

I just want to drop a quick note to anyone who may search google for a problem I ran into.A nested jQuery mobile list can be populated with one knockoutjs observable array. The only caveat I've seen thus far is that you need to use the foreach data-binding twice. Once on the outer list and again for the inner "list" or page.
